Unit 6 - Control Structures

Lesson: For Loops

For loops are used to repeat a block of code a specific number of times. Here's an example:


for i = 1, 10 do
    print("Count: " .. i)
end

This will print numbers from 1 to 10.

Exercise

Create a countdown from 15 to 0 using a for loop. Your code should look something like this:


for i = 15, 0, -1 do
    -- Your code here
end
